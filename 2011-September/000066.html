Hallo Olaf,

momentan werden Dialog in einer festen Größe ausgegeben. Das führt unter 
Umständen bei zu großen Standardschriften (siehe auch 
http://www.jverein.de/forum/viewtopic.php?f=5&t=711&p=2781) zu Problemen.

Siehst du eine Möglichkeit in Jameica eine generelle Lösung für Dialoge 
einzubauen. Ich stelle mir das so vor, dass der Benutzer den Dialog in 
die benötigte Größe zieht. Beim nächsten Aufruf wird der Dialog dann 
wieder in der letzten Größe dargestellt.
