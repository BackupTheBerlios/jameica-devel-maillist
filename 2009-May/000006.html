Hallo Olaf,

ich habe der Klasse SearchInput einen neuen Konstruktor hinzugefügt. 
Damit ist es möglich den angezeigten Initialtexte (Suche...) individuell 
einzustellen.


  /**
   * Erzeugt eine neue Such-Box.
   */
  public SearchInput()
  {
    this(Application.getI18n().tr("Suche..."));
  }

  /**
   * Erzeugt eine neue Such-Box mit Vorgabe des searchstrings
   */
  public SearchInput(String searchstring)
  {
    super();
    this.search = searchstring;
  }


Würdest du die Änderung bitte ins CVS einchecken?
